# aidoo-test

## 项目架构
项目采用**nacos**注册中心，**zuul**网关做一个简单Spring Cloud。zuul做鉴权网关过滤流量，token采用**jwt**生成与校验。使用**knife4j**做接口文档输出，结合**mybatis-plus**做user的CRUD。
采用DockerFile部署项目到个人服务器。
